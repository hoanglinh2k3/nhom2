## Mô tả đề tài:
- Xây dựng trang web bán đồ decor, bao gồm các chức năng như thêm, sửa, xóa, và hiển thị thông tin sản phẩm.
- Sử dụng ngôn ngữ lập trình javascript để phát triển hệ thống.
- Áp dụng mô hình lập trình đa tiến trình (multi-process) để xử lý các yêu cầu từ người dùng.
- Sử dụng các kỹ thuật quản lý tiến trình và giao tiếp giữa các tiến trình (IPC) để đảm bảo hệ thống hoạt động hiệu quả và ổn định.
- Tích hợp các tính năng bảo mật để bảo vệ thông tin sản phẩm khỏi các truy cập trái phép.
- Cung cấp giao diện dòng lệnh (CLI) để người dùng có thể tương tác với hệ thống một cách dễ dàng.

## Hướng dẫn sử dụng:
1. Cài đặt môi trường phát triển javascript trên máy tính của bạn.
2. Tải mã nguồn của dự án từ kho lưu trữ.
3. Biên dịch mã nguồn bằng cách sử dụng trình biên dịch javascript (ví dụ: node).
4. Chạy chương trình từ dòng lệnh và làm theo hướng dẫn để quản lý thông tin sản phẩm.
5. Sử dụng các lệnh tương ứng để thêm, sửa, xóa, và hiển thị thông tin sản phẩm.

## Yêu cầu hệ thống:
- Hệ điều hành: Linux hoặc Windows với môi trường phát triển javascript.
- Trình biên dịch javascript (ví dụ: node).
- Quyền truy cập dòng lệnh (CLI).
